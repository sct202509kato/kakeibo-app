# 家計簿アプリ

React + TypeScript + Vite で作成した、個人利用向けの家計簿アプリです。

URL:https://kakeibo-app-ubx5.onrender.com


## 概要
- 収入・支出の登録
- 取引履歴の表示
- カテゴリ別集計
- LocalStorage によるデータ永続化

## 技術スタック
- React
- TypeScript
- Vite

## セットアップ
```bash
npm install
npm run dev
```

## 補足
学習・ポートフォリオ目的で作成しています。
